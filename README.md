## Gulp-Aurora
A Gulp boilerplate for most of JSish framework projects.

### What is included
* Jade
* Sass
* Connect (Livereload)
* Debug

### How to use it
```
npm install
gulp
```
For build production pack for deploy:
```
gulp prod
```
